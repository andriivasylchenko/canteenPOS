# IBM MobileFirst Platform Foundation JSONStore Plugin
This plugin enables the JSONStore feature in a Cordova application already installed with the plugin, cordova-plugin-mfp. It works the same way as the JSONStore feature that is included in IBM MobileFirst™ Platform Foundation. 

JSONStore allows for persistent, always-available storage, and access of JSON documents, even when there is no network connection. 

Refer to the documentation below for more information. 

## Installation
Using the MobileFirst CLI plugin add command:

    	mfp cordova plugin add cordova-plugin-mfp-jsonstore
    
## Dependencies
- cordova-plugin-mfp

## Supported Platforms
- Android
- iOS
